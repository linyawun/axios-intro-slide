## Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) and [pnpm](https://pnpm.io/) installed on your system.

### Installation

1. Install dependencies:
```bash
pnpm install
```

2. Start the development server:
```bash
pnpm dev
```

3. Open your browser and visit:
   - [http://localhost:3030](http://localhost:3030)

## Development

- Edit the [slides.md](./slides.md) file to modify the slide content
- Changes will be reflected in real-time

## Additional Commands

```bash
# build slides for production
pnpm build

# preview the production build
pnpm preview
```
